# Проект To-Do созданный в практических целях

## Как установить?

1. Клонируйте репзиторий и после перейдите к каталогу
```
git clone <repo url>
```
2. Установите зависимоссти
```
python3 -m pip install -r requirements.txt
```
3. Примените миграций
```
python3 manage.py migrate
```
4. Добавьте переменнные окружения (см. ниже)

## Переменные окружения

Создайте в корне файл `.env` и впишите переменные окружения:

- SECRET_KEY= Секретный ключ джанго
- DEBUG= True or False (default=True)
- ALLOWED_HOSTS= Хосты прослушивания (default=None)
